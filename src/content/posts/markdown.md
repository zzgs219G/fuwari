# 这是一级标题 (h1)

段落之间可以用一个空行来分隔。

这是第二段落。你可以使用 _斜体_，**加粗**，或者是 `代码字体`。无序列表长这个样子：

- 第一项
- 第二项
- 第三项

> 这里是引用区块。
> 就像这样写。

## 这是二级标题 (h2)

这是一个有序列表：

1. 第一项
2. 第二项
3. 第三项

下面是一个代码示例：

    # 让我再重复一遍 ...
    for i in 1 .. 10 { do-something(i) }

正如你猜到的，缩进 4 个空格就是代码块。顺便说一下，除了缩进，你更可以用反引号来包裹代码：

```
define foobar() {
    print "Welcome to flavor country!";
}
```

（这样写能让你复制粘贴更方便）。你可以指定语言名称，系统会自动为你添加语法高亮：

```python
import time
# 快速数到十！
for i in range(10):
    # (但也不能太快)
    time.sleep(0.5)
    print(i)
```

### 这是三级标题 (h3)

现在来看一个嵌套列表：

1. 首先，准备这些食材：

    - 胡萝卜
    - 芹菜
    - 扁豆

2. 烧开一些水。

3. 把所有东西倒进锅里，然后遵循以下算法：

        找到木勺
        打开锅盖
        搅拌
        盖上锅盖
        把木勺小心地平衡在锅柄上
        等待 10 分钟
        返回第一步（或者煮熟后关火）

    千万不要碰到木勺，否则它会掉下来。
