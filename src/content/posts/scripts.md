## 快速使用
执行以下命令即可一键加载所有脚本功能：

```bash
source <(curl -fSL https://raw.githubusercontent.com/zzgs219G/scripts/main/fzgit.sh)
```
#带下载进度条
```bash
source <(curl -fkSL -# https://raw.githubusercontent.com/zzgs219G/scripts/main/fzgit.sh)
```
